# Installation

faqCombo requires:
- python
- colorama
- requests
- datetime
- codecs
- isort
- webbrowser
- getpass

To install run these comands:

<h2>Windows: </h2>

```sh
▶ git clone https://github.com/JustFaQ/faqCombo.git
```
```sh
▶ cd faqCombo
▶ requirement_install.bat
▶ run_script.bat
```

<h2>Other: </h2>

```sh
▶ sudo apt install git
```
```sh
▶ git clone https://github.com/JustFaQ/faqCombo.git
```
```sh
▶ cd faqCombo
▶ requirement_install.sh
▶ run_script.sh
```
